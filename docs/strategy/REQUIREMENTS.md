# 要件定義書 (REQUIREMENTS) - Ver.2.0

## 1. プロジェクトの概要
### 1.1 ブランド名
**Zen Conditioning / Kukan**
「身体を通して心を整える」をコンセプトに、禅の知恵と現代コンディショニング（整体、鍼灸、トレーニング、坐禅）を統合したアプローチを提供するスタジオの予約管理システム。

### 1.2 目的
洗練されたブランドイメージを損なわない、シンプルで直感的な予約体験を提供し、トレーナー側の稼働管理とトレーニー側の予約利便性を向上させる。

## 2. ターゲットユーザー (ペルソナ)
### 2.1 トレーニー (顧客)
- 30〜50代の多忙なビジネスパーソン。
- 自身の心身のバランスを整えるために定期的に通う。
- 予約や変更をスマホからストレスなく完了させたい。

### 2.2 トレーナー (代表・スタッフ)
- 施術・指導のプロフェッショナル。
- 自身の稼働時間を柔軟に設定し、予約状況をリアルタイムで把握したい。
- 顧客のカルテやプラン利用状況を一元管理したい。

## 3. ビジネスルール
### 3.1 セッション・予約単位
- **セッション構成**: 55分セッション ＋ 5分インターバル（合計60分）。
- **予約受付単位**: 顧客は30分または60分刻みのスロットから選択可能。

### 3.2 予約受付・変更期限
- **予約可能期間**: 30日前 〜 1日前 24時まで。
- **変更・キャンセル期限**: 〜 1日前 24時まで。

### 3.3 営業時間
- **定休日**: 水曜日
- **標準営業時間**: 9:00 〜 23:00 (祝日含む)
- **設定単位**: 30分単位でトレーナーごとに稼働枠を設定可能。

### 3.4 料金・商品体系
- **チケット・プラン**:
  - 600分チケット: 150,000円 (税込)
  - 月4回コース: 40,000円 (税込)
  - 都度課金 60分セッション: 17,000円 (税込)
- **諸費用**: 入会金 20,000円、登録料 10,000円
- **オプション**: 円皮鍼 100円/枚、鍼 2,100円

## 4. MVP (Phase 1) の範囲
- トレーナーによる稼働枠（Availability）の登録。
- 顧客による30分単位での空き枠検索と予約。
- 1日前24時までの期限を厳守した予約・キャンセルロジック。
- メールアドレス/パスワードによるセキュアなユーザー認証。
- 基本的なプロフィール（名前、メール、電話番号、ロール）管理。

※ チケット残回数の自動管理や自動決済は Phase 2 以降とする。
