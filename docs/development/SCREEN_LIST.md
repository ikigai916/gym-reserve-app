# 画面一覧

## トレーニー（顧客）向け画面

### 1. ログイン/新規登録画面
- **URL**: `/login.html`（既存）
- **機能**:
  - 新規登録
  - ログイン
  - ロール選択（トレーニー/トレーナー）
- **状態**: 実装済み（ロール選択機能も実装済み）

### 2. 予約一覧/カレンダー画面
- **URL**: `/index.html`（既存）
- **機能**:
  - カレンダー表示
  - 予約可能日時の確認
  - 予約状況の表示
- **状態**: 実装済み（プラン・メニュー対応は未実装）

### 3. 予約作成画面
- **URL**: `/reservation/new`（新規）
- **機能**:
  - プラン選択（所有プランから選択）
  - メニュー選択（大カテゴリ）
  - 日時選択
  - 残りセッション数の表示
  - 予約確認・作成
- **状態**: 未実装

### 4. マイページ
- **URL**: `/mypage`（新規）
- **機能**:
  - 個人情報の表示・編集
  - 所有プラン一覧
  - プラン有効期限・残り回数表示
  - 予約履歴
  - カルテ履歴へのリンク
- **状態**: 部分実装（既存issue #7）

### 5. マイ予約一覧画面
- **URL**: `/my-reservations`（新規、またはindex.htmlに統合）
- **機能**:
  - 自分の予約一覧
  - 予約詳細（日時、メニュー、プラン）
  - 予約キャンセル
- **状態**: 未実装（既存issue #5）

### 6. カルテ閲覧画面
- **URL**: `/charts`（新規）
- **機能**:
  - 自分のカルテ履歴一覧
  - カルテ詳細（大カテゴリ、中カテゴリ、備考）
  - 日付・カテゴリでのフィルタリング
- **状態**: 未実装

### 7. プラン購入画面
- **URL**: `/plans/purchase`（新規）
- **機能**:
  - プラン一覧表示
  - プラン詳細・価格表示
  - プラン選択
  - 決済画面への遷移
- **状態**: 未実装

### 8. 決済画面
- **URL**: `/payment`（新規）
- **機能**:
  - 購入プランの確認
  - 決済方法の選択
  - 決済情報の入力（クレジットカードなど）
  - 決済実行
  - 決済完了画面
- **状態**: 未実装

### 9. 決済履歴画面
- **URL**: `/payment/history`（新規）
- **機能**:
  - 決済履歴一覧
  - 決済詳細（プラン、金額、日時、ステータス）
  - 領収書のダウンロード（オプション）
- **状態**: 未実装

## トレーナー向け画面

### 1. トレーナーログイン画面
- **URL**: `/trainer/login`（新規）
- **機能**:
  - トレーナー専用ログイン
  - ロール認証
- **状態**: 未実装

### 2. 予約管理画面（ダッシュボード）
- **URL**: `/trainer/dashboard`（新規）
- **機能**:
  - 本日の予約一覧
  - 今週の予約一覧
  - 予約ステータス管理
  - 予約検索・フィルタリング
- **状態**: 未実装

### 3. 予約詳細画面
- **URL**: `/trainer/reservations/:id`（新規）
- **機能**:
  - 予約詳細情報
  - 顧客情報
  - カルテ記録画面への遷移
  - 予約ステータス変更
- **状態**: 未実装

### 4. カルテ記録画面
- **URL**: `/trainer/charts/new`（新規）
- **機能**:
  - 予約選択
  - 大カテゴリ選択
  - 中カテゴリ入力・選択
  - 備考入力
  - カルテ保存
- **状態**: 未実装

### 5. カルテ編集画面
- **URL**: `/trainer/charts/:id/edit`（新規）
- **機能**:
  - 既存カルテの編集
  - カルテ削除
- **状態**: 未実装

### 6. 顧客管理画面
- **URL**: `/trainer/customers`（新規）
- **機能**:
  - 顧客（トレーニー）一覧
  - 顧客検索
  - 顧客詳細（プラン状況、予約履歴、カルテ履歴）
- **状態**: 未実装

### 7. プラン管理画面
- **URL**: `/trainer/plans`（新規）
- **機能**:
  - プラン一覧
  - プラン作成・編集・削除
  - プラン価格設定
- **状態**: 未実装

### 8. メニュー管理画面
- **URL**: `/trainer/menus`（新規）
- **機能**:
  - メニュー一覧（カテゴリ別）
  - メニュー作成・編集・削除
  - メニュー所要時間設定
- **状態**: 未実装

### 9. 決済管理画面
- **URL**: `/trainer/payments`（新規）
- **機能**:
  - 決済履歴一覧（全顧客）
  - 決済詳細
  - 決済ステータス管理
  - 決済検索・フィルタリング
  - 売上統計（オプション）
- **状態**: 未実装
