# ジム予約管理システム

ジムの時間枠を予約・管理するシンプルなWebアプリケーションです。

## 機能

- 予約可能な時間枠の表示
- 予約の作成（名前、日付、時間枠）
- 自分の予約一覧の確認
- 予約のキャンセル

## 技術スタック

- **フロントエンド**: React + Vite
- **バックエンド**: Node.js + Express
- **データストレージ**: JSONファイル

## セットアップ

### 必要な環境

- Node.js (v18以上推奨)

### インストール手順

1. リポジトリをクローン（またはこのディレクトリに移動）

2. バックエンドのセットアップ
```bash
cd backend
npm install
```

3. フロントエンドのセットアップ
```bash
cd ../frontend
npm install
```

## 起動方法

### 開発環境

1. バックエンドサーバーを起動（ターミナル1）
```bash
cd backend
npm run dev
```
バックエンドは http://localhost:3001 で起動します。

2. フロントエンドサーバーを起動（ターミナル2）
```bash
cd frontend
npm run dev
```
フロントエンドは http://localhost:3000 で起動します。

3. ブラウザで http://localhost:3000 を開く

## 使い方

1. **予約する**
   - 「予約する」タブを選択
   - お名前を入力
   - 予約したい日付を選択
   - 予約可能な時間枠をクリックして予約

2. **マイ予約を確認**
   - 「マイ予約」タブを選択
   - 予約時に使用した名前を入力
   - 自分の予約一覧が表示されます

3. **予約をキャンセル**
   - 「マイ予約」タブで予約一覧を表示
   - キャンセルしたい予約の「キャンセル」ボタンをクリック

## データの保存場所

予約データは `backend/data/reservations.json` に保存されます。

## 時間枠

デフォルトでは以下の時間枠が利用可能です：
- 09:00-10:00
- 10:00-11:00
- ...
- 21:00-22:00

（9時から22時までの1時間単位）

## 制約事項

- 1つの時間枠に1名まで予約可能
- 過去の日付には予約不可
- 予約は名前ベースで管理（認証なし）
